# Contributing

This distribution uses an issue-first maintenance model.

## Accepted

After a GitHub repository is published, open an issue for a reproducible workbook defect, broken link, documentation ambiguity, or accessibility problem. Use only fictional sample data and include the release version and spreadsheet application version.

## Not accepted

- Pull requests that replace or modify workbook source files.
- Pull requests containing generated workbooks, ZIP files, paid-edition files, or alternative downloads.
- Real customer, order, payment, receipt, contact, credential, or internal business data.
- Requests to add macros, banking access, account access, scraping, tracking, or upload behavior.
- Promotional, affiliate, or unrelated submissions.

Source-file pull requests will remain closed unless the maintainer explicitly changes this policy in a later release. Documentation suggestions should also begin as an issue so scope and privacy can be checked before any change is prepared.
